# zpy-vue

## Project setup
```
npm install express --save
npm install --save socket.io
npm install
npm install -g @vue/cli
npm install vue-smooth-dnd
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run ZPY
```
npm run build
node server/app.js
```
